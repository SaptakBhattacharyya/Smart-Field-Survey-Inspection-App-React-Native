# Smart Field Survey Inspection App

A comprehensive field survey and inspection suite built with React Native and Expo. This application provides field agents with tools to conduct surveys, capture data, and monitor field activities efficiently.

## Features

- User Authentication and Profile Management
- Dashboard with Quick Actions and Analytics
- Dark Mode Support
- Integrated Camera for Survey Evidence
- Location Tracking and GPS Status
- Contacts Management
- Clipboard Integration for Quick Data Entry
- Drawer and Tab-based Navigation

   ```bash
   npm install
   ```

